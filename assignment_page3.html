Task 1

Задание 1 Основы CSS.
  1. Придумать визуальное оформление страницы из задания 1* (цвета, шрифты, фон итд) и реализовать его средствами CSS. СSS-правила должны подключаться из внешнего файла.
  2. Тег iframe. Во-первых, необходимо ознакомиться с тегом информацию можно найти тут: https://habr.com/ru/post/488516/ю Во-вторых, необходимо добавить этот тег на свою страницу. Например, видео.
  3. Дополнить страницу базовыми META-тегами. (Информация тут: https://www.ashmanov.com/education/articles/meta-tegi/ )
* Создать страницу-портфолио со следующей информацией: ФИО студента; номер группы; название и ссылка на страницу кафедры; название и ссылка на сервер ВУЗа.
